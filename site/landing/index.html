AI agent workflow

Use the Same KB Workflow Across Common AI-Enabled IDEs

The template stores prompts, governance, revision state, and maintenance rules as plain Markdown, so the workflow stays editor-agnostic for common AI-enabled IDEs and editors. Start with the same documents whether you work in VS Code, Cursor, Windsurf, JetBrains IDEs, or another agent-capable environment.

Before the agent writes

Read Template Index, How To Use, Metadata Schema, and Repository Revision State first.

Choose the right prompt

Use Prompting Guide for strategy, Prompt Pack for ready-made prompts, and Agent Manual for operating rules.

Keep drift under control

Compare the stored baseline with git HEAD before broad maintenance, then run the maintenance loop or patch flow.

Tooling

CLI Preview — kb

A local CLI for initializing, maintaining, and health-checking downstream KB instances. No server required.

Install & run
# One-off
npx @williamduong/kb help

# Global install
npm install -g @williamduong/kb
kb help

# Init a new KB
kb init --mode private-git
  • help — list available commands
  • init --mode private-git|tracked — bootstrap a new KB
  • show / hide — toggle private-git visibility
  • sync [--accept-baseline] — pull latest template changes
  • update [--accept-baseline] — apply version patches
  • doctor [--json] [--strict] — validate KB health
  • test — run template self-checks

Get started

Getting Started In 3 Steps

Do the setup first, pick one prompt block below, then validate and start development.

Step 1 | Install + bootstrap
# Install or run one-off
npm install -g @williamduong/kb
kb help

# Bootstrap a new knowledge base
kb init --mode private-git

# Check health after your first fill pass
kb doctor --strict
  1. Step 1. Install + bootstrap: install the CLI, run kb help, then initialize with kb init --mode private-git.
  2. Step 2. Choose one prompt block below: read Prompting Guide first, then copy the prompt or command that matches your goal.
  3. Step 3. Validate and start development: run kb doctor, open the KB in your AI IDE, and continue from 00-start-here/finalization-plan.md.
